## What to build

Adaptive layout for the `Game.vue` view using Tailwind CSS breakpoints. On desktop (≥768px), the chat panel sits to the right of the board — both visible simultaneously. On mobile (<768px), the chat panel stacks below the board. The board remains centered and fills available space. The lobby also adapts: room list takes full width on mobile, wider on desktop. Final visual polish: consistent spacing, Tailwind color palette for dark/light theme, proper button styling, input field styling for chat, responsive font sizing for piece characters and board labels.
