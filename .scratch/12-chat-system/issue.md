## What to build

In-game chat via `ChatPanel.vue` — a scrollable message list and a text input. Players type a message and press Enter (or click Send). The message is sent over WebSocket as `{ type: "chat", text }`. The server adds `sender` (clientId), `timestamp`, and `color` (red/black), and broadcasts `{ type: "chat", message: { sender, text, timestamp, color } }` to both players. Messages are color-coded: Red player's messages appear with a red indicator, Black player's with a black/dark indicator. Messages auto-scroll to the bottom.

## Acceptance criteria

- [ ] `ChatPanel.vue` shows a scrollable message list and a text input + Send button
- [ ] Typing text and pressing Enter sends `{ type: "chat", text }` over WebSocket
- [ ] Server enriches the message with `sender`, `timestamp`, `color` and broadcasts to both players
- [ ] Both players receive and display the message

- [ ] Messages show relative timestamp (e.g., "just now", "2m ago")
- [ ] Message list auto-scrolls to the latest message
- [ ] Chat does not appear in the lobby — only during gameplay
- [ ] Integration test: two clients, send chat from Red, verify Black receives it with correct color
