## What to build

Create `src/server/protocol.ts` with a `ServerMessage` discriminated union type covering all 22 server→client message variants. This is the centralized type that replaces ad-hoc `Record<string, unknown>` message shapes scattered across the codebase.

Also standardize `chat` from `data.type === "chat"` to `action: "chat"` in the protocol (client→server messages already use `action` for most commands).

**Message variants:**
- `connected`, `roomCreated`, `roomUpdate`, `lobbyUpdate`
- `gameStart`, `boardUpdate`, `gameEnd`
- `error`, `kicked`
- `chat`, `drawOffered`, `drawDeclined`
- `rematchState`, `opponentReconnected`, `roomReclaimed`
- `pong`
