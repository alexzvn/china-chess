## What to build

During gameplay, either player can click "Offer Draw" or "Resign". Resign immediately ends the game — the opponent wins. Draw offer sends `{ type: "drawOffer" }` to the server, which relays it to the opponent. The opponent sees "Opponent offers a draw" with Accept / Decline buttons. Accept → game ends as a draw, both see "Game ended — Draw". Decline → game continues, "Draw declined" notification. The draw offer and resign buttons appear in the game UI (e.g., below the board or in PlayerInfo). After game end via resign/draw, the same status bar with "Back to Lobby" appears (as in #11).
