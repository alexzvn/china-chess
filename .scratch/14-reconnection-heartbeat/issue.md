## What to build

Elysia's built-in WebSocket ping/pong heartbeat detects disconnects: server pings every 30s, client must pong within 60s or is marked disconnected. When a game player disconnects, the server starts a 60-second grace period — the game pauses (no moves accepted), the opponent sees "Opponent disconnected, waiting...". If the player reconnects within 60s (by opening a new WebSocket and re-identifying with the roomId + clientId), the game resumes from the saved state. If the 60s expires, the disconnected player forfeits — opponent wins.

The Vue client auto-reconnects every 3 seconds on disconnect, showing "Reconnecting..." status. On successful reconnect, it re-joins the room by sending `{ action: "rejoinRoom", roomId, clientId }`.
