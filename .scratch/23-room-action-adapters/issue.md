## What to build

Create `src/server/actions/room.ts` with 5 action adapters. Each adapter receives a context object, calls the Room module, reads the room state, and returns an `ActionResult` with notifications.

**Adapters:**
- `handleCreateRoom(ctx: NoRoomActionContext)` — calls `createRoom(clientId)`, sends `roomCreated` + `roomUpdate` to creator, broadcasts lobby
- `handleJoinLobby(ctx: NoRoomActionContext)` — sends `lobbyUpdate` with all waiting rooms
- `handleJoinRoom(ctx: RoomActionContext)` — calls `joinRoom(roomId, clientId)`, sends `roomUpdate` to both players, broadcasts lobby
- `handleKickPlayer(ctx: RoomActionContext)` — calls `kickPlayer(roomId, clientId)`, sends `kicked` to kicked player, `roomUpdate` to host, broadcasts lobby
- `handleLeaveRoom(ctx: RoomActionContext)` — calls `leaveRoom(roomId, clientId)`, sends appropriate messages based on whether host or playerB left

**Context types:**
```typescript
interface RoomActionContext {
  roomId: string
  clientId: string
  room: Room
  from?: Position
  to?: Position
  send: (clientId: string, message: ServerMessage) => void
  broadcastLobby: () => void
}

interface NoRoomActionContext {
  clientId: string
  send: (clientId: string, message: ServerMessage) => void
  broadcastLobby: () => void
}

type ActionResult = 
  | { kind: 'error'; message: string }
  | { kind: 'ok'; notifications: Notification[] }

type Notification =
  | { kind: 'send'; clientId: string; message: ServerMessage }
  | { kind: 'broadcastLobby' }
```
