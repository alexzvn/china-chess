## What to build

The Elysia server exposes a WebSocket endpoint at `/ws`. The Vue client opens a WebSocket connection, receives a server-assigned `clientId` (nanoid 7 chars), and sends messages. The server echoes back a confirmation. Opening two browser tabs both show "Connected" with their respective `clientId`. No rooms or game logic yet — just the WebSocket handshake and echo ping/pong.

## Acceptance criteria

- [ ] `ws://localhost:3000/ws` accepts WebSocket upgrade via Elysia
- [ ] On connection, server generates `clientId` via `nanoid(7)` and sends `{ type: "connected", clientId }` to the client
- [ ] Client displays "Connected: <clientId>" in the Vue UI
- [ ] Server sends back `{ type: "pong" }` when client sends `{ type: "ping" }` — client displays "Echo received"
- [ ] Opening two browser tabs shows two distinct `clientId` values, each "Connected"
- [ ] Disconnecting (close tab) logs a disconnect message on the server (structured JSON log)
