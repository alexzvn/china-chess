## What to build

Players can join a room by navigating to `/room/:id` or clicking a room in the lobby. When the second player joins, both players see a confirmation screen showing their roles (Player A / Player B). Both players must click "Start Game" — the server then randomly assigns Red/Black colors, sets the room status to `playing`, broadcasts `{ type: "gameStart", yourColor: "red"|"black" }` to each player, and navigates both to the `Game.vue` view. Colors are random — no player choice.

## Acceptance criteria

- [ ] Clicking a room in `Lobby.vue` navigates to `/room/:id` via Vue Router
- [ ] On join, server sets `playerB` to the joining client's `clientId`, broadcasts `{ type: "roomJoined", roomId }` to both players
- [ ] After join, both clients see a "Waiting" / confirmation screen with "Start Game" button

- [ ] Client receives `yourColor` and displays it (e.g., "You are Red" / "You are Black")
- [ ] After game start, `RoomCard` for this room no longer appears in the lobby list
- [ ] Red moves first — server's `turn` field is set to `"red"` on game start
- [ ] Integration test: create room, join room, both confirm, verify game started with correct colors
