## What to build

When the game ends via checkmate or stalemate, the server broadcasts `{ type: "gameEnd", result, winnerColor, reason }`. The `Game.vue` frontend shows a status bar below the board with the result message (e.g., "Checkmate! Red wins" or "Stalemate! Black wins") and a "Back to Lobby" button. Clicking "Back to Lobby" sends `{ action: "leaveRoom" }` over WebSocket and navigates back to `/`. On the server side, the room is destroyed when both players leave or after game end. Pieces can no longer be clicked after game end.
