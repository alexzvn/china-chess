## What to build

`Board.vue` renders a 9×10 Chinese Chess grid with all visual elements: flat-colored squares, river text "楚河漢界" spanning the board between rank 4 and 5, palace diagonal lines drawn via CSS in the 3×3 areas on each side, traditional point dots at standard positions (8 per half). `Piece.vue` renders individual pieces as Unicode Chinese characters inside circular backgrounds with an inner ring (CSS concentric circles), red text on cream for Red pieces, black text on cream for Black pieces. The board scales responsively to fit any viewport using Tailwind responsive utilities and CSS `vmin`/`vw` units. Static initial board position with all 32 pieces.

## Acceptance criteria

- [ ] Board is a 9 columns × 10 rows CSS grid
- [ ] Alternating light/dark flat colors per square (no wood texture)
- [ ] River text "楚河漢界" visible and centered between rank 4 and 5
- [ ] Palace diagonal lines visible in the 3×3 areas on each side (ranks 0-2 and 7-9)
- [ ] Traditional point dots at standard positions (top, middle, bottom, corners of each half)
- [ ] Pieces rendered as circular backgrounds with inner ring
- [ ] Red pieces: red text (#cc0000 or similar) on cream/lighter background
- [ ] Black pieces: black text (#111 or similar) on slightly darker cream background
- [ ] Initial board shows all 32 pieces at standard Chinese Chess starting positions
- [ ] Board scales to fit viewport — no scrollbars on any screen size (desktop, tablet, mobile)
- [ ] Component test via Vitest: board renders 90 cells + 32 pieces in starting position
